# peronal_finance
<h1>Introduction</h1>
<p>This project aims to provide any user with the opportunity to control their perosnal finances so they can better control their money. The foucs of this project is:</p>
<ul>
<li>Track expenses</li>
<li>Set and follow up on monthly budget per category (rent, food, entertainment, etc)</li>
<li>Set and follow savings goals</li>
<li>Set and follow debt payments goals</li>
<li>Set and follow investment goals</li>
</ul>
